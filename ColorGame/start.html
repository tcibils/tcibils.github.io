Bienvenue dans mon jeu des couleurs !

Je me suis amusé à faire ce petit jeu pour maîtriser un peu mieux le HTML, le CSS, et surtout pour créer un projet qui soit visuellement plaisant :-) J'espère qu'il vous plaira.

Dans ce jeu, je vous présenterai d'abord des couleurs avec leurs noms dans un bouton. Vous devrez les retenir, puis cliquer sur leur nom. Ensuite, je vous représenterai ces couleurs avec deux options : un nom de couleur, ou "faux". Si ce nom est le bon, cliquez dessus. Sinon, cliquez sur "faux".

Vous inquiétez pas, ça commence facilement avec un tutoriel :-)

Bon jeu,
